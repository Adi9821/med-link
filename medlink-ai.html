🔄 Workflow

How MEDLINK-AI
Works in 5 Phases

From first registration to AI-powered consultation — here's the complete patient journey.

1️⃣
📝

Phase 1: Registration & Health ID Generation

Onboarding

Patient signs up with basic details (name, DOB, Aadhaar/phone). The system auto-generates a unique Health ID (MED-YEAR-XXXX format) and creates a QR code. Auth0 handles secure authentication — no password stored in plaintext.

POST /api/register → { healthId: "MED-2024-7829", qrToken: "eyJ..." }
2️⃣
🗂️

Phase 2: Medical Profile Setup

Profile

Patient uploads existing records (PDF reports, X-rays, prescriptions). Google Cloud Vision OCR extracts text from images. AI engine categorizes documents and populates the structured health profile — blood group, allergies, chronic conditions, medications.

📄 PDFs
🖼️ Images
💊 Rx
3️⃣
🔐

Phase 3: Consent & Access Control

Security

Patient grants access to specific doctors with time-limited tokens. Every access event is logged immutably. Emergency QR bypasses login but only reveals critical info — not full history. All data encrypted at rest with AES-256.

🔒 Access Layers
Layer 1: Patient (full access)
Layer 2: Authorized Doctor (partial, time-limited)
Layer 3: Emergency (blood type + allergies only)
4️⃣
🏥

Phase 4: Doctor Consultation & AI Summary

AI-Powered

Doctor scans QR or searches by Health ID. AI generates a 3-minute summary of the patient's entire history. Doctor can update prescriptions, add consultation notes, or upload new reports. All changes sync to patient dashboard instantly.

🤖 AI Summary: "Patient has Type 2 Diabetes (diagnosed 2019), currently on Metformin 500mg. Last HbA1c: 7.2% (Aug 2024). No known drug allergies. Recommended: Continue current medication, schedule 3-month follow-up."
5️⃣
🔄

Phase 5: Continuous Sync & Security Audit

Ongoing

All updates sync in real-time via Google Cloud Pub/Sub. Automated anomaly detection flags unusual access patterns. Monthly security reports delivered to patient. Compliance audits ensure HIPAA, ISO 27001 standards are continuously met.
